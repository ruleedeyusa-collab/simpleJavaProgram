# simpleJavaProgram
a simple java program

## Steps to Compile and Run the Program
1.) Compile the Program
 - Navigate to the project folder : cd HelloWorldProject
 - Compile : javac -d out src/com/example/helloworld/HelloWorld.java
 - This will create a compiled class file in: out/com/example/helloworld/HelloWorld.class

2.) Run the Program
- Use the fully qualified package name:java -cp out com.example.helloworld.HelloWorld
